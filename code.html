The Code for FIZZ BUZZ


                    //Launch Function
                    function startFizzBuzz() {
                        let fizzNumber = document.getElementById("FizzNumber").value;
                        let buzzNumber = document.getElementById("BuzzNumber").value;
                        
                        fizzNumber = Number.parseInt(fizzNumber);
                        buzzNumber = Number.parseInt(buzzNumber);

                        if (typeof fizzNumber === "number" && typeof buzzNumber === "number" && 
                        fizzNumber > 0 && fizzNumber <= 100 && 
                        buzzNumber > 0 && buzzNumber <= 100) {

                            //Logic Function
                            let fizzBuzzArray = chooseFizzOrBuzzNumber(fizzNumber, buzzNumber);

                            //Display Function
                            displayFizzBuzz(fizzBuzzArray);
                        }
                        else {
                            alert("You have to put a value between 1 and 100")
                        }

                    }

                    //Logic Function
                    function chooseFizzOrBuzzNumber(fizzNumber, buzzNumber) {

                        let numbersArray = [...Array(100).keys()].map(k => k + 1);

                        let fizzBuzzArray = numbersArray.map(number => 
                            number % fizzNumber === 0 && number % buzzNumber === 0 ? "FizzBuzz" :
                            number % fizzNumber === 0 ? "Fizz" :
                            number % buzzNumber === 0 ? "Buzz" :  number
                        );

                        return fizzBuzzArray;
                    }

                    //Display Function
                    function displayFizzBuzz(fizzBuzzArray) {

                        document.getElementById("table").classList.remove("d-none");

                        let table = document.getElementById("results");
                        table.innerHTML = "";
                        let row;

                        for (let i = 0; i < fizzBuzzArray.length; i++) {
                            if (i === 0 || i % 5 === 0 ){
                                row = table.insertRow();

                            }
                            let cell = row.insertCell();
                            cell.innerHTML += `${fizzBuzzArray[i]}`;
                        }
                    }

The code is separated in 3 functions. Each one has its purpose and so, it only contains what it has to execute.

startFizzBuzz()

As its name implies, it launches and controls the application. Here, I haven't used ES6 and instead I have used the traditional if/else. This is for readability purpose only. It works the other way also but, i will show how it appears:

It is more compact, but not so readable. There is another problem too, when the numbers don't satisfy the conditions, the variable has a null value. It can be used to our advantage, in the display function, but we don't need it, as the function won't launch the way it is coded.

chooseFizzOrBuzzNumber(fizzNumber, buzzNumber)

This function generates and returns an array. To make the array I could have chosen a For Loop, but I have chosen another method, more concrete, more readable and faster. The array is done with the array generator, but the array is empty. So the spread operator is used and the keys of the array (the indexes) to generate the array, but the array starts at 0 and ends with 99, so the map function is used to sum 1 to each number. Then another map function is used to generate the fizzBuzzArray.

displayFizzBuzz(fizzBuzzArray)

Its porpuse is to display the array into a table. To make this first we make the table visible, removing a class from it. Then we grab the tbody component where the results will go (I have put the name table, even it is not, to make it easier to read). The innerHTML of the table is reseted and we use a for loop to make the different rows and cells and display the results of the array. Probably, there is a ES6 better way to display the results (replacing the For Loop), but I haven't found the way yet.
